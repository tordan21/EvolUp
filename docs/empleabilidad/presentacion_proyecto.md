# Presentación del Proyecto — EvolUp

## ¿Qué es?

EvolUp es una aplicación de escritorio en Java para gestionar objetivos personales y hábitos. Permite crear objetivos, asociarles hábitos concretos, registrar si se han cumplido cada día y ver el progreso a lo largo del tiempo.

## ¿Qué problema resuelve?

Mucha gente sabe qué quiere mejorar pero no lleva ningún registro de si lo está cumpliendo. Una lista de tareas no sirve para esto porque no está pensada para hábitos a largo plazo.

EvolUp cubre eso: defines una meta, le asocias hábitos y ves día a día si los estás cumpliendo.

## ¿Para quién está pensado?

Para cualquier persona que quiera mejorar en algún área de su vida de forma ordenada: aprender un idioma, mejorar hábitos de salud, estudiar con más constancia, etc.

## Tecnologías utilizadas

- **Java 17** — lógica de la aplicación y menú de consola
- **MySQL 8.0** — almacenamiento de datos
- **JDBC** — conexión entre Java y MySQL
- **Maven** — gestión de dependencias
- **XML / XSD** — exportación de datos y validación de estructura

## ¿Qué he aprendido desarrollándolo?

He aprendido a conectar una aplicación Java con una base de datos real usando JDBC, a diseñar un modelo de datos con relaciones entre tablas, y a organizar el código en capas (modelo, DAO, utilidades) en lugar de poner todo junto. También he trabajado con XML y validación de esquemas, y he usado Git para gestionar el historial del proyecto.
