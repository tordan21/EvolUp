# Reflexión Personal

## ¿Qué he aprendido?

He aprendido más de lo que esperaba. No tenía claro cómo iba a conectar todo entre sí. Ahora entiendo cómo una aplicación accede a una base de datos, cómo se organizan las clases para que el código no sea un caos, y cómo se valida información tanto en SQL como en XML.

También he aprendido a usar Git de forma continua, no sólo para subir el proyecto al final.

## ¿Qué se me ha dado mejor?

La parte de bases de datos. Diseñar las tablas, las relaciones y escribir las consultas SQL. Una vez entendí cómo funcionaban las claves foráneas y los JOINs, el resto fue más sencillo.

## ¿Qué me ha costado más?

Dos cosas: la organización general del proyecto y la parte de programación en Java. Al principio no tenía claro cómo dividir el código, qué ponía en cada clase ni cómo se conectaban entre ellas. He tenido que replantear cosas varias veces hasta que empezó a tener sentido.

## ¿Qué mejoraría?

El proyecto se queda sólo en consola. Me gustaría haberle añadido una interfaz gráfica, aunque sea simple, para que fuera más usable. También me falta añadir lógica para calcular rachas automáticamente en lugar de hacerlo sólo con consultas SQL.

## ¿Qué pasos quiero dar?

Quiero seguir aprendiendo los diferentes lenguajes de programación hasta sentirme cómodo con ellos, y especializarme en analizar datos para saber en base a qué toman decisiones las empresas. Más adelante me gustaría investigar más cómo se aplica la IA en proyectos concretos. El objetivo a largo plazo es crear aplicaciones que hagan algo útil con datos reales.
