# Perfil Profesional

Estudio el ciclo de Desarrollo de Aplicaciones Multiplataforma (DAM) en su primer curso. Me metí en esto porque quería saber cómo se construía una aplicación desde el principio.

Lo que más me llama la atención dentro del mundo tecnológico son los datos y la inteligencia artificial. Me parece interesante cómo a partir de información aparentemente desordenada se pueden sacar conclusiones útiles o construir sistemas que aprendan. No sé todavía todo lo que implica, pero es el área hacia la que quiero ir.

Ahora mismo estoy aprendiendo Java, Python, SQL, XML y conceptos de orientación a objetos. También he trabajado con Maven y JDBC para conectar una aplicación Java con una base de datos MySQL.

A largo plazo me gustaría especializarme en análisis de datos o desarrollo de aplicaciones que usen IA de forma práctica. Lo que me motiva es crear algo que resuelva un problema concreto — no proyectos por el hecho de hacerlos, sino cosas que tengan utilidad real.
